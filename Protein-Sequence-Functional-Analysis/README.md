# Protein Sequence Analysis and Functional Characterization

This project focuses on comparative protein sequence analysis and evolutionary study using bioinformatics tools.

## Tools Used
- BLASTp
- Clustal Omega
- Phylogeny.fr
- PyMOL

## Analysis Performed
- Identified homologous proteins across primate species using BLASTp.
- Conducted multiple sequence alignment for comparative sequence analysis.
- Built phylogenetic trees to study evolutionary relationships.
- Visualized protein structure and identified EF-hand calcium-binding domains.

## Outcome
The analysis helped understand evolutionary conservation and structural features of the studied protein.
