# Biological Sequence Analyzer using Python

This project is a simple Python program developed to analyze biological sequences.

## Features
- Calculates sequence length
- Counts amino acids in protein sequences
- Calculates molecular weight
- Calculates GC content for DNA sequences

## Tools Used
- Python
- Basic bioinformatics concepts

## About the Project
This project was created as part of my learning in bioinformatics and Python programming. It shows how simple scripts can help analyze biological sequence data.
