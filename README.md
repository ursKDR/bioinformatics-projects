
 Bioinformatics Projects

This repository contains academic bioinformatics projects related to protein sequence analysis and computational biology.

Projects

1. GLS Protein Analysis in Osteosarcoma
- Protein sequence retrieved from UniProt
- Homology search using BLASTp
- Multiple sequence alignment using Clustal Omega
- Structure prediction using AlphaFold

2. Protein Sequence Functional Analysis
- Comparative protein analysis using BLAST
- Phylogenetic analysis using Phylogeny.fr
- Protein structure visualization using PyMOL

 3. Biological Sequence Analyzer using Python
- Python script for analyzing FASTA sequences
- Calculates sequence length, amino acid composition, molecular weight, and GC content
