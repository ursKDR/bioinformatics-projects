# GLS Protein Sequence Analysis in Osteosarcoma

This project analyzes the GLS (Glutaminase) protein associated with osteosarcoma metabolism using bioinformatics tools.

## Tools Used
- UniProt
- BLASTp
- Clustal Omega
- AlphaFold
- Ensembl VEP

## Analysis Performed
- Retrieved GLS protein sequence (669 amino acids) from UniProt.
- Performed BLASTp search to identify homologous protein sequences.
- Conducted multiple sequence alignment (MSA) using Clustal Omega.
- Studied predicted 3D structure using AlphaFold.
- Evaluated potential variant effects using Ensembl Variant Effect Predictor (VEP).

## Outcome
This analysis helped understand conserved regions and structural properties of the GLS protein related to osteosarcoma.
